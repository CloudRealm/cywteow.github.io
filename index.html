About

This is a kodi addon that allows you to play drama from langlangbay.org in your android/android tv

Installation

1) Install kodi from play store
2) Download the zip file and transfer into your android/android tv
3) Open Kodi -> Add-on -> Addon browser -> Install from zip file -> select the zip file you downloaded (Ensure your kodi can install from unknown sources in system settings)

How to see chinese words in kodi

Go to Kodi system setting -> Interface -> Skin -> Fonts -> Arial based
